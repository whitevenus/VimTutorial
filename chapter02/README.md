# 高级编辑和文本浏览

## 插件安装
**插件目录**: `~/.vim/pack/plugins/start/`,每次安装新插件时,都将插件对应的github仓库克隆在以下位置即可。

设置插件自动加载:
```vim
packoaddall              " 加载所有插件
silent! helptags ALL     " 为所有插件加载帮助文档
```
